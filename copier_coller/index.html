Comment copier et coller un fichier ou un dossier sur un ordinateur ?

Avec la souris

  1. Sélectionner le ou les éléments : Utiliser le bouton gauche de la souris pour sélectionner le fichier ou le texte à copier.
  2. Copier : Cliquer avec le bouton droit de la souris sur la sélection et choisir l'option "Copier" dans le menu contextuel.
  3. Coller : Accéder à l'emplacement où vous souhaitez coller le fichier ou le texte, cliquer avec le bouton droit et choisir l'option "Coller" dans le menu contextuel.

Avec un clavier

Combinaisons de touches

Copier un ou plusieurs éléments

Après la sélection des éléments à copier, effectuer la combinaison de touches pour copier:

Système d'exploitation Combinaison
Windows Ctrl+C
Distributions Linux Ctrl+C
macOS Command+C

Coller un ou plusieurs éléments

Une fois que vous êtes à l'emplacement où vous souhaitez coller les éléments, utiliser la combinaison de touches appropriée:

Système d'exploitation Combinaison
Windows Ctrl+V
Distributions Linux Ctrl+V
macOS Command+V

Copier et coller en ligne de commande

Avec Windows

copy chemin_source\nom_fichier.extension chemin_destination\

Avec Linux ou macOS

cp chemin_source/nom_fichier.extension chemin_destination/
